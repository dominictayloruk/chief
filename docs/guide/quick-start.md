# Quick Start

Get Chief running in under 5 minutes.

::: tip Prerequisites
You need [Claude Code CLI](https://github.com/anthropics/claude-code) installed and configured.
:::

## Installation

```bash
brew install minicodemonkey/chief
```

## Create Your First PRD

```bash
chief init
```

This creates a `.chief/` directory with a sample PRD to get you started.

## Run the Loop

```bash
chief
```

Watch as Claude autonomously works through your user stories.
